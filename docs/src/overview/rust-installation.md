# 开发环境配置
下面演示一下Raven开发环境配置的流程：
## 安装Rust
编译和运行Raven引擎需要使用rust，请访问官方进行数据
```
https://www.rust-lang.org/zh-CN/
```
## 拉取最新代码
**使用GitKraken** 或者是 **GitHub Desktop** 工具进行代码拉取。地址如下：
```
https://github.com/E72UJ/Raven.git
```
